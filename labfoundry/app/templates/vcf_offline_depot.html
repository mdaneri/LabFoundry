VCF Offline Depot

Reset VCFDT package

This removes the staged VCF Download Tool package, saved Broadcom download token or activation code, and generated software depot metadata from LabFoundry desired state, and disables all download profiles. It does not mutate appliance runtime files until global appliance apply removes the staged and active tool trees.

VCF Offline Depot

Refresh software depot ID

This submits the VCF Offline Depot apply unit. LabFoundry validates the desired state, stages the VCF Download Tool, runs vcf-download-tool configuration generate --software-depot-id, and records the result in the apply job.
